# NOU Digital Learning & Assessment System

A web-based Digital Learning and Assessment System developed for Nalanda Open University using Java Spring Boot and MySQL.

## Features

- Student Registration and Login
- Admin Dashboard
- Online Examination System
- Question Bank Management
- Study Material Upload
- Result Management
- Email Notification System
- Responsive Bootstrap UI
- Course Management
- Feedback and Enquiry System

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Java Spring Boot
- MySQL
- STS / Eclipse IDE

## Modules

### Admin
- Manage Students
- Upload Study Materials
- Add Questions
- Schedule Tests
- Publish Results

### Student
- Register/Login
- Access Study Materials
- Attend Online Exams
- View Results
- Submit Feedback

## Database

- MySQL Database
- Tables:
  - studentinfo
  - admininfo
  - enquiry
  - questionbank
  - testinfoes
  - testresult

## Project Objective

The system modernizes distance education by providing secure and efficient digital learning, examination, and assessment facilities.
